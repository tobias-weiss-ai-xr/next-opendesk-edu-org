# Notes

Notes (also referred to as Docs) is a lightweight note-taking application designed for quick text notes, markdown documents, and simple document editing. It priorits is speed and simplicity over feature bloat, making it ideal for jotting down ideas, meeting notes, and short-form content. As a base openDesk CE component, it is stable and production-ready.

## Key Features

- **Fast editing**: Minimal, distraction-free editor that loads instantly in the browser.
- **Markdown support**: Write and render markdown content with live preview.
- **Simple organization**: Create, edit, and delete notes without complex folder structures.
- **Lightweight footprint**: Low resource requirements, suitable for environments with limited capacity.

## Integration with openDesk Edu

Notes integrates with openDesk Edu through Keycloak-based SSO using SAML 2.0 and OIDC. Users access the editor directly from the unified Nubs portal. It deploys as a modular Helm chart with minimal resource overhead.
