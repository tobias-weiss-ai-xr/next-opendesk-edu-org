# Draw.io

Draw.io (also known as diagrams.net) is a dedicated diagramming tool for creating architecture diagrams, flowcharts, UML diagrams, network diagrams, and other visual documentation. It supports a wide range of shapes, connectors, and export formats. Draw.io is an education-specific Beta component in openDesk Edu.

## Key Features

- **Extensive shape libraries**: Built-in libraries for UML, BPMN, network, AWS, Azure, GCP, and general-purpose shapes.
- **Export formats**: Export diagrams to PDF, VSDX, PNG, SVG, and XML for sharing and further editing.
- **Import support**: Import existing diagrams from VSDX, Lucidchart, Gliffy, and other formats.
- **Collaboration**: Real-time collaborative editing when connected to a storage backend.
- **Diagram templates**: Start from templates for common diagram types like org charts, flowcharts, and ER diagrams.

## Integration with openDesk Edu

Draw.io integrates with openDesk Edu through Keycloak-based SSO using SAML 2.0 and OIDC. Users access the editor from the unified Nubus portal. It deploys as a modular Helm chart as an-education-specific Beta component. Draw.io provides a dedicated diagramming option alongside the diagrams.net integration in CryptPad and the whiteboard option in Excalidraw.
