# 聊天

Element 是基于 Matrix 协议的即时通讯应用，集成了 Nordeck 教育小部件。支持实时消息、聊天室和文件共享，适用于师生日常沟通和教学讨论。

## 核心功能

- **实时消息**: 支持一对一和群组聊天，消息端到端加密
- **聊天室**: 可创建主题聊天室，按课程或项目组织讨论
- **文件共享**: 在聊天中直接共享文件和图片
- **Nordeck 教育小部件**: 集成投票、问卷调查等教育互动工具
- **音视频通话**: 支持一对一和群组语音/视频通话

## 与 openDesk Edu 的集成

聊天作为 openDesk CE 基础组件，通过 Keycloak SSO（SAML 2.0 / OIDC）实现统一身份认证，用户登录一次即可访问所有服务。通过统一门户可直接进入聊天界面。采用模块化 Helm Chart 部署，与平台其他组件共享基础设施。
