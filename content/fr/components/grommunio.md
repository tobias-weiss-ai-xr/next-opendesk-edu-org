# Grommunio

Grommunio est un groupware compatible Microsoft 365 qui offre une expérience familière pour les utilisateurs habitués à l'écosystème Microsoft. Grâce à son support d'ActiveSync 16.1, il permet une synchronisation native avec les appareils mobiles (iOS, Android) et les clients de messagerie classiques. Grommunio utilise MariaDB comme base de données.

## Fonctionnalités clés

- **Compatibilité Microsoft 365**: Interface et fonctionnalités familières pour les utilisateurs
- **ActiveSync 16.1**: Synchronisation push native avec smartphones, tablettes et Outlook
- **Messagerie et calendrier**: E-mail complet avec gestion d'agenda partagé
- **Base de données MariaDB**: Stockage fiable et éprouvé pour les données groupware

## Intégration avec openDesk Edu

Grommunio est un composant en version Beta pour le contexte éducatif. Il se déploie via Helm et s'intègre au portail unifié avec authentification Keycloak en SSO (SAML 2.0 / OIDC). Il vient en alternative à OX App Suite et SOGo pour les établissements qui privilégient la compatibilité avec les clients Microsoft.
